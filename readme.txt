== Installation ==

1. Unzip padvertisement-management.zip
2. Upload all the files to the /wp-content/plugins/padvertisement-management
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. For customizing the plugin's settings, click on Advertisement in Wordpress left admin menu and click on Settings.

== Frequently Asked Questions ==
= What does this plugin do? =
This plugin creates ad like post/page in Advertise Mgmt. It Helps you to display your advertise in your website easily.

= Do I need to design the Podamibe Advertisement Management? = 
You won't need to take any burden regarding the design as since we have already provided Desigin Option on settings which you can choose as per your requirement.


= Where can I find the documentation for the plugin? =
Once you install the plugin , you can check some general documentation about how to use the plugin in the "How to use" panel of the plugin's settings page.

= How can I display the Podamibe Advertisement Management? =
Use this for individual "[pam post_id='ads_id']" and for bulk "[pam_bulk pad_id='ads_id,ads_id,...']" shortcode to display in content. For template use &lt;?php echo do_shortcode('[pam post_id='ads_id']');?&gt;. Use Widget for sidebar.
